Como Funciona Queremos saber los servicios que brinda para ayudarle!

Ajustadoati es una plataforma para búsqueda de productos y servicios en tiempo real usando la ubicación geográfica y mensajería instantánea, la misma servirá de base para cualquier persona o empresa que ofrezca algún servicio. El modo de funcionamiento es bastante sencillo ya que permite a los proveedores (persona, empresa) registrarse con las categorías en la que presta sus servicios y descargar la aplicación (Por ahora APK-TEST Android) para recibir las peticiones y responder las mismas. Tanto en el site como en la aplicación móvil se encuentra el formulario de registro en el cual se carga la información básica y se selecciona la categoría o las categorías si presta más de un servicio. Los usuarios que solo la tendrán para realizar búsquedas no necesitan seleccionar categoría. La búsqueda se realiza desde la web o la aplicación móvil colocando el producto o servicio a buscar y seleccionando la categoría, al realizar la petición la plataforma se encargará de enviar la misma a todos los proveedores registrados en la categoría, en el mapa de búsqueda se podrán ver los mismos. Aunque se mostrarán en el mapa todos los proveedores que aplican para la categoría de la petición, el mensaje se le enviará sólo a los que están conectados al momento pero de igual forma se mostrará sus datos de contacto para que el usuario se comunique más adelante, si algún proveedor responde la petición automáticamente se refleja en el mapa del usuario y podrá interactuar directamente con él a través de mensajería instantánea (Llamada próxima fase).
